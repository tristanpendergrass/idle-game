# Development

You must download [Lamdera](https://dashboard.lamdera.app/docs/download).

```
$ npm run build-css -- --watch
```

And in a different tab:

```
$ lamdera live
```

# Deployment

```
$ npm run build-css
$ lamdera check
$ lamdera deploy
```

Read more about deploying Lamdera [here](https://dashboard.lamdera.app/docs/deploying).
